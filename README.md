# EF Bets

A progressive web app (PWA) for sports betting, built with React, TypeScript, and Vite.

## Features

- Mobile-first design with bottom navigation
- PWA support - installable on iOS and Android
- Offline capability with service worker
- Dark theme UI

## Pages

- **Home** - Balance overview, quick actions, live events
- **Bets** - View and track your bet history
- **Profile** - User stats and settings

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
- React Router
- vite-plugin-pwa
